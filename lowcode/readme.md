## 概念定义
低代码或零代码是通过可视化的拖拽代替手写代码来构建应用，低代码适合开发者提效，零代码让非技术人员也能搭建简单应用，常用于表单、审批流程、数据看板等场景，快速满足业务需求。
- 可视化 canvas
- 拖拽 好理解 工作流，程序流

## Aisuda
Aisuda是基于低代码、零代码的开发平台，提供可视化编辑器、组件库、模板市场、插件市场等功能，支持自定义开发，可以快速搭建应用。

## 核心

  - 组合显示区域（drop） 网页、agent、工作流（n8n，dify）
